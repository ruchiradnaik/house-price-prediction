# House Price Prediction

This project uses machine learning to predict house prices based on various features.

## Setup Instructions
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using `pip install -r requirements.txt`.
4. Train the model by running `python model/train_model.py`.
5. Start the Flask app using `python app.py`.
6. Send a POST request to `/predict` with the required features to get predictions.

## Features
- Predict house prices based on input features.
